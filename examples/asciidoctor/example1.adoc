= Example Gantt Diagram

== Project Plan

This example shows how to include a Gantt chart in Asciidoctor using the `gantt` block.

.Plano de execução do projecto
[gantt, target=project-plan.svg]
----
period: "week"
activities:
  1, "Gestão de Projecto", duration=13
  2, "Planning"
    2.1, "Discovery", duration=2
    2.2, "Design", duration=3, dependencies=2.1
  3, "Delivery"
    3.1, "Implementation", duration=6, dependencies=2.2
    3.2, "QA", duration=2, dependencies=3.1
  3.25, "Release Prep", duration=1, dependencies=3
  3.3, "Training", duration=1.5, dependencies=3.2, notBefore=12
  4, "End of project", dependencies=3.2
----

The chart above should render as an SVG and be embedded into the PDF output.
