== Project Plan

This example shows how to include a Gantt chart in Asciidoctor using the `gantt` block.

[gantt, target=project-plan.svg]
----
period: "week"
activities:
  1, "Planning"
    2, "Discovery", duration=2
    3, "Design", duration=3, dependencies=2
  4, "Delivery"
    5, "Implementation", duration=6, dependencies=3
    6, "QA", duration=2, dependencies=5
  7, "End of project", dependencies=6
----

The chart above should render as an SVG and be embedded into the PDF output.
